An **AI-powered Food & Health application** that provides **personalized dietary recommendations** based on user health data such as **age, BMI, lifestyle, and medical conditions**.
The system simulates decision-making similar to **professional dietitians and pediatricians**, delivering customized meal plans, nutrition insights, and health tracking dashboards.

## 🚀 Features

* 🧠 AI-based personalized diet recommendations
* 📊 Health dashboard with analytics and progress tracking
* 🍽️ Automatic meal plan generation (breakfast, lunch, dinner, snacks)
* 📈 BMI and calorie requirement calculation
* 📝 Meal tracker to log daily food intake
* 📅 Meal history with nutrition insights
* ⚕️ Disease-specific diet recommendations (diabetes, thyroid, PCOS, etc.)
* 👶 Pediatric diet support for children
* 💧 Water intake and nutrition monitoring
* 🔄 Dynamic diet suggestions based on user inputs

---

## 💡 Workflow

1. User enters personal details (age, height, weight, lifestyle)
2. System calculates BMI and nutrition requirements
3. User provides medical conditions and dietary preferences
4. AI engine analyzes data using nutrition rules
5. Personalized meal plan is generated
6. User tracks meals using Meal Tracker
7. Dashboard visualizes progress and nutrition insights
